# Apple M2 GPU — Measured Properties

All numbers from real probe runs on this device.

## Basics
- **Device:** Apple M2, unified memory
- **SIMD width:** 32 threads
- **Max threadgroup size:** 1024 threads
- **Max threadgroup memory:** 32 KB
- **Max buffer length:** 4096 MB
- **Recommended working set:** 5461 MB
- **Dispatch overhead:** ~2.7 us
- **GPU timer resolution:** ~5.7 us minimum measurable dispatch

## Memory Bandwidth
- **DRAM bandwidth:** ~81 GB/s
- **Threadgroup memory bandwidth:** ~1571 GB/s aggregate
- **Texture read bandwidth:** ~77 GB/s
- **Buffer read bandwidth:** ~136 GB/s

## Cache Hierarchy
- **4 KB:** ~57 ns
- **16 KB:** ~76 ns
- **64 KB:** ~104 ns
- **256 KB:** ~115 ns
- **1024 KB:** ~114 ns
- **4096 KB:** ~174 ns
- **16384 KB:** ~232 ns
- **32768 KB:** ~232 ns
- **SLC associativity:** 16-way
- **TG memory bank count:** 32

## Coalescing
- **Stride 1 (coalesced):** ~60 GB/s
- **Stride 16 (non-coalesced):** ~14 GB/s
- ~4.2x penalty for non-coalesced access

## Compute Throughput
- **FP32 ALU:** ~2806 GFLOPS
- **FP16 ALU:** ~2856 GFLOPS (same as FP32 — no packed fp16)
- **INT32 ALU:** ~619 GIOPS (~4.5x slower than FP32)
- **Shader cores:** ~52 (inferred from saturation sweep)

## SIMD Operations
- **SIMD shuffle:** ~377 Gshuffles/s, ~1507 GB/s effective
- **SIMD reduction (simd_sum):** ~65 Greductions/s

## Atomics
- **Uncontested:** ~19.7 Gatomics/s
- **Full contention (1 counter):** ~0.7 Gatomics/s
- Contention penalty: ~28x

## Synchronization
- **Barrier cost (32-thread TG):** ~75 ns
- **Barrier cost (1024-thread TG):** ~82 ns
- **TG memory RAW latency (1 thread):** ~87 ns per store-barrier-load cycle
- **TG memory RAW latency (256 threads):** ~87 ns

## Register File
- **Spill boundary:** between 24 and 32 simdgroup_float8x8 accumulators

## Pipeline
- **FMA throughput (independent):** ~513 GFLOPS
- **FMA latency (dependent chain):** ~308 GFLOPS
- Ratio ~1.7x suggests ~2-cycle FMA pipeline

## Instruction Cache
- **Small kernel (4 FMA/iter):** ~969 GFLOPS
- **Medium kernel (32 FMA/iter):** ~1019 GFLOPS
- **Large kernel (64 FMA/iter):** ~719 GFLOPS
- Large variant shows ~29% drop — icache pressure

## TLB
- **4 pages:** ~69 ns
- **8 pages:** ~57 ns
- **16 pages:** ~57 ns
- **32 pages:** ~57 ns
- **64 pages:** ~57 ns
- **128 pages:** ~68 ns
- **256 pages:** ~135 ns
- **512 pages:** ~153 ns
- **1024 pages:** ~96 ns

## MMA (simdgroup_multiply_accumulate)
- **Latency (dependent):** ~1.0 ns/MMA
- **Throughput (4 independent):** ~0.5 ns/MMA
- ~1.9x speedup with independent accumulators

## Dispatch
- **Direct dispatch:** ~3.5 us
- **Indirect dispatch:** ~6.6 us
- **Indirect overhead:** ~3.1 us

## Readback Latency
| Size | GPU | Readback | Total |
|------|-----|----------|-------|
| 4 KB | 4 us | 243 us | 248 us |
| 64 KB | 5 us | 229 us | 234 us |
| 1024 KB | 12 us | 270 us | 282 us |
| 16384 KB | 195 us | 257 us | 453 us |

## Concurrent Execution
- **Kernel A alone:** 3.580 ms
- **Kernel B alone:** 3.337 ms
- **Both together:** 5.991 ms
- **Overlap:** 13%
- GPU runs independent kernels concurrently across cores.
